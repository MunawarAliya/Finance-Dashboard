# Finance Dashboard

A simple React-based finance dashboard that tracks income, expenses, and balance.

## Features

- View total balance, income, and expense
- Expense breakdown chart by category
- Search and filter transactions
- Add and delete transactions in admin mode
- Transaction data persisted in `localStorage`

## Usage

1. Open `index.html` in a browser.
2. Switch to `Admin` role to add or delete transactions.
3. Use the search box to find transactions by category.
4. Use filters to show only income or expense transactions.
